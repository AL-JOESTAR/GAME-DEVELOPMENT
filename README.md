# Clean The Island

**Clean The Island** adalah game bertema lingkungan yang dikembangkan sebagai bagian dari **Project Game Development** dengan menerapkan konsep **Game Development Life Cycle (GDLC)**.

Game ini mengajak pemain untuk membersihkan sebuah pulau dari berbagai sampah yang berserakan agar lingkungan kembali bersih dan nyaman.
